## 📚 Дополнительные материалы

### Характерные слова по странам:
| Страна | Характерные слова |
|--------|-------------------|
| Мексика | Güey, chido, ¿qué onda? |
| Аргентина | Che, boludo, vos |
| Испания | Tío, guay, coño, mola |
| Куба | Asere, ¿qué bolá? |
| Колумбия | Parcero, bacano |
| Чили | Po, cachai, weon |

### Особенности произношения:
- Испания: distinción (Z = θ)
- Аргентина: LL = Ш, voseo
- Карибы: проглатывание согласных
- Мексика: чёткое произношение
